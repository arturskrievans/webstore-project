Website's Functionality:

This website supports all CRUD operations via creating accounts and products, displaying their information as well as updating and deleting them.

By uploading a lot of products to the shop (at least 5) you can additionally see pagination in effect to make navigation process easier. Filtering by price and date is also possible. Products that are sold cannot be further modified and are simply marked as "sold" with an eye-catching visual effect.

All the necessary information is stored in the database models such that the users don't have to reupload it on server and session restarts.

Basic security protocols like password hashing (werkzeug.security), user authentication and link validation are followed. Meaning, no sensitive information is stored in plain text anywhere, you cannot access different accounts or perform unauthorized operations by manipulating the URL parameters.

As you can see by the Global Chat page, this website does not only use the HTTP communication protocol, but also takes the advantage of WebSockets - request and receive information with persistent connection.
In short, the HTTP protocol is used for all CRUD methods where new data is first received from a form and only then displayed. WebSockets are used for a live global chat room where we don't want the user to have to reload the page to see new messages. Nevertheless, the last 50 messages are still saved in the database and rendered on page load so users can view some message history.

What Was Used For The Implementation?

All the backend was implemented using Flask - a lightweight Python web framework.

For database models I used SQLAlchemy - a Python SQL toolkit that supports nearly all SQL based operations.

For fully functional WebSocket implementation and more interactive front-end JavaScript was also applied.

To secure bidirectional communication for the live chat, I additionally used Socket.IO - a Python and JavaScript supported library for real-time web apps.

Libraries like werkzeug, os, uuid were used to make the website more secure and robust.

To structure, display and style the content HTML + css were also utilized.
